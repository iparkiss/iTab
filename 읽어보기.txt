■ v 1.5.3
1. 아이콘 편집 기능을 추가하다.
2. 설정창에 아이콘 모양을 슬라이드로 모서리 둥글기를 조절하게 하다.
3. 아이콘 수정모드에서 아이콘 크기를 조절하는 슬라이드를 추가하다.
4. 아이콘 수정모드에서 아이콘 배경색을 선택하도록 컬러 팔레트를 추가하다.
5. 아이콘 수정모드에서 아이콘 타일 테두리 표시 체크박스를 추가하다.
6. 아이콘 수정모드에서 ESC를 누르면 편집 모드까지 종료되는걸 아이콘 수정모드만 종료되도록 수정하다.
7. 페이지 인디케이터를 < >에서 원형(점)으로 변경하다.
8. 페이지 전환 시 인디케이터에 애니메이션을 추가하다.
9. 설정 메뉴 배경 섹션에 배경 이미지 블러 기능을 추가하다.
10. 아이콘 드래그 시 조금만 이동했다가 놓아도 원치 않은 위치로 이동하는 드래그/드롭의 불안정한 부분 개선하다.
11. 아이콘 드래그 시 애니메이션을 조금 더 부드럽게 개선하다.

■ v 1.5.2
1. 아이콘 위에 마우스 올렸을때 마우스 포인터 모양 변경하다.
2. 아이콘 수정모드 시 아이콘 위에 마우스가 올라가면 펜 아이콘을 표시하게 하다.
3. 아이콘 수정모드 시 애니메이션을 없애다.
3. 수정모드에서 나오는 방법을 [완료]버튼 클릭에서 빈 바탕 클릭으로 변경하다.
4. 빈 바탕을 클릭하는 경우 항상 검색창이 포커스를 가지게 하다.

■ v 1.5.1
1. 검색창이 포커스를 잃은 상태에서 브라우저가 포커스를 잃었다가 다시 포커스를 받을 경우 
    검색창이 포커스를 가져가면서 최근목록 박스가 열리는 문제를 해결하다.
2. 검색창의 최근목록 박스가 열린 상태로 브라우저가 포커스를 잃을 경우 목록 박스가 닫히지 않는 문제를 해결하다.
3. 새 탭 바탕의 대표 썸네일을 클릭해서 배경 이미지를 변경할 경우 최근 이미지 박스가 바로 닫히지 않고
    대표 썸네일을 다시 클릭하거 빈 바탕을 클릭하여 닫도록 수정하다.
4. 설정창에 바로가기 아이콘들을 북마크로 내보내기 기능을 추가하다.

■ v 1.5.0
1. 검색창의 검색어를 chrome.storage.local에 20개 저장되도록 보완하다.
2. 새로운 검색어가 추가될 경우 가장 오래된 검색어가 삭제되도록 하다.
3. 검색창에 단어 입력 시 최근 검색어에 등록된 검색어가 있을 경우 
4. 검색창 아래에 목록박스로 보여주고 그 아래에 추천 검색어를 보여주도록 하다.
5. 최근 검색어가 없을 경우에는 추천 검색어만 보여주도록 하다.
6. 최근 검색어는 목록 박스에서 전체삭제, 개별삭제 가능하도록 하다.
